GAME-CRAFTERS


This websites provides you a collection of diffrent types of games that can be implemented using Python.
It is with the initiative of teaching interesting Python libraries and interesting functions of Python in a fun manner.
As we all have played many offline games in our childhood that we can never forget.
Along with that we all have played games on computers.

Let's see how we can implement some of them and learn Python.



EGG - CATCHER GAME

Eggs Catcher is a classic game where the goal is to catch as many eggs as possible. whenever we catch an egg then our score increases and if we miss three eggs then we will lose the game.

THE BOUNCING BALL GAME

This Bouncing Ball Game is to hit all of the targets as many times as possible in order to achieve the highest possible score.


ANIMATED FIDGET


A fidget spinner is a toy that consists of a ball bearing in the center of a multi-lobed flat structure made from metal or plastic designed to spin along its axis with pressure.Making it digital and increasing the spin when the mouse clicks.

CAR GAME


You’ll be required to control your vehicle using arrow keys (left, right, and up) and If you crash into another car you’ll lose so be careful. Let’s start building it!!!!





BUBBLE SHOT GAME


This game is a simple shooting game. While playing the game, all you have to do is just use your mouse for pointing to the balloons and use the right-click mouse button to shoot the balloon. There is a simple and clean GUI for easy gameplay. Here, the player has to use the mouse to shoot the balloon

STICK MAN RUN GAME

The player can control the stick man via keyboard bindings(Left Arrow Key to turn left, Right Arrow Key to turn right, Space Bar to jump). The game is very simple you just need to jump across the platform and enter the door as fast as you can. The stick man will continue to move when you turn in any side
